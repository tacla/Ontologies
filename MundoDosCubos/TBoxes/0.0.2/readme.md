Inclui os axiomas seguintes sobre as propriedades:
* inversas: estáSobre &lt;-&gt; estáSob
* temMassa: funcional (um objeto tem um valor único de massa em kg)
* transitiva: b -&gt; estáSobre -&gt; a -&gt; estáSobre -&gt; m, logo b -&gt; estáSobre -&gt; m
* simétrica: d &lt;- estáAoLadoDe -&gt; e
* assimétrica: c -&gt; estáAFrenteDe -&gt; g (g não pode estar a frente de c)
* reflexiva e simétrica: c -&gt; temMesmaCor -&gt; c
* irreflexiva: a -&gt; estáAFrenteDe -&gt; a (não é possível se for irreflexiva)
* cadeia de propriedades: b -&gt; estáSobreOCubo -&gt; a -&gt; estáSobreAMesa -&gt; m então b estáNoNível2DaMesa m.</Literal>
